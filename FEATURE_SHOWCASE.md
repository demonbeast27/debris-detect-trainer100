### Using the Heatmap API

```javascript
// Initialize heatmap
const heatmap = new DebrisHeatmap('heatmapCanvas', imageElement);

// Load detection data
const detections = [
    {
        class_name: "plastic",
        confidence: 0.92,
        bbox: [100, 100, 200, 200],
        center: { x: 150, y: 150 }
    }
];
heatmap.setDetections(detections);

// Show/hide
heatmap.toggle();

// Get statistics
const stats = heatmap.getStats();
console.log(`Hotspots: ${stats.hotspots}`);
```
